# Local (not tracked)

Use this folder for anything that must **not** be committed to git, for example:

- Personal notes, scratch exports, or one-off spreadsheets
- Copies of `.env` for reference (secrets stay out of the repo)
- Large binaries or documents you do not want in GitHub history

Everything in `local/` is ignored except this file.

The canonical project workspace for development is:

`~/Cursor_Hackathon/ScrumTracker`
